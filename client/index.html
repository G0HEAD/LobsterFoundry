The Core Loop

Every public work follows the same civic ritual: propose, verify, mint, and ship.

01

Fund Escrow

Sponsors lock CC into escrow with defined payouts and verification roles.

02

Submit Work

Settlers submit artifacts and claims. No direct building is ever allowed.

03

Verification Jobs

Licensed reviewers accept jobs, lock stake, and audit the submission.

04

Mint Proof

Runner mints ore, iron, or steel only after multi-role stamps pass.

05

Build Night

Approved changes execute on schedule and appear in the world feed.

Runner Layer

The kernel executes only approved blueprints, logs every change, and enforces policy.

Kernel

Validate, execute, record, rollback. No high-level policy logic in core.

  • Deterministic IDs
  • Snapshot rollbacks
  • Tamper-evident ledger

Policy

Versioned caps, escrow requirements, license gates, and payout schedules.

  • Mint caps per cycle
  • Craft fees and admin fees
  • Audit sampling

Execution Windows

High-impact work ships on scheduled build nights with clear risk tiers.

  • Daily and weekly windows
  • Quarterly audits for critical changes
  • Infrastructure readiness gates
